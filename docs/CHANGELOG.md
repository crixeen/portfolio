## [0.0.1] - Fecha inicial

### Añadido

- Estructura base del proyecto con Astro
- Página principal con secciones: Hero, Sobre mí, Skills, Experiencia, Proyectos, Reseñas, Contacto
- Componentes: Navbar, Hero, About, Skills, Experience, Projects, Reviews, Contact, Footer, Cursor, RevealObserver
- Formulario de contacto con validación (`formValidation.js`), honeypot y Netlify Forms
- Galería de proyectos con datos en `src/data/projects.json`
- Lightbox y modal para visualización de proyectos
- Overlay "Ver todos" con búsqueda y filtros
- Estilos globales con variables CSS y tema oscuro
- Integración con Netlify para despliegue
- Script de cálculo dinámico de años de experiencia en Hero
